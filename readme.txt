=== DFM WP Google Tag Manager ===


Includes tags (code snippets) from Google Tag Manager into your site.
Wordpress is not allowing to insert any code after body tag, this plugin insert google tag code by jQuery prepend function.

== Description ==
This plugin deploy the Google Tag Manager code snippet into a WP site.

== Installation ==
1. Upload folder `dfm-wp_google_tag_manager` into `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Enter your public container id in 'Settings' --> â€˜DFM WP Google Tag Manager' menu in wordpress
